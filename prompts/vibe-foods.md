Build a full single-page landing page for a luxury omakase sushi restaurant called **Takumi**. 
Use semantic HTML and CSS in a single file.

## Visual Identity
White marble texture backgrounds, gold (#C9A84C) accents, and ivory/off-white surfaces. 
Typography should pair an elegant serif display face (e.g. Cormorant Garamond or Playfair Display) 
with a clean sans-serif body. Minimal, refined, and high-end — think Nobu meets a Michelin-starred 
tasting menu.

## Sections (in order)

1. **Hero** — Full-viewport, dramatic. Restaurant name "Takumi" in large display type, a short 
evocative tagline, and a gold "Reserve a Table" CTA button.

2. **Menu Highlights** — Showcase 4–6 signature omakase dishes with a name, short poetic 
description, and price.

3. **Our Story** — A brief, elegant paragraph about the chef's philosophy and the restaurant's 
origins.

4. **Reservations** — A simple booking form with fields for name, date, time, party size, and a 
submit button styled in gold.

5. **Gallery** — A grid of placeholder images (use solid marble/dark-toned CSS placeholders if no 
images are available).

6. **Hours & Location** — Clean typographic layout with address, hours by day, and an embedded or 
placeholder map.

7. **Reviews** — 3 pull-quote testimonials from fictional guests, displayed elegantly.

## Design Details
- **Palette:** ivory white `#FAFAF7`, warm gold `#C9A84C`, deep charcoal `#1A1A1A`, marble grey `#E8E4DC`
- No bright colors; no gradients except subtle marble texture via CSS
- Sticky nav — restaurant name left-aligned, anchor links right-aligned
- Smooth scroll between sections
- Subtle fade-in animations on scroll
- Fully responsive down to mobile
